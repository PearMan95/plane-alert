How it works

Plane Alert polls the airplanes.live API in the background. When an aircraft matching one of your alerts enters your radius for the first time, you get a desktop notification. You only get one notification per aircraft per visit — once it leaves your radius and comes back, you'll be notified again. Use the on/off toggle in the header to pause and resume tracking at any time.

Live tab

Matches onlyHide aircraft that don't match any active alert

Detail paneelClick any aircraft to see registration, type, altitude, speed, route, squawk and heading. Use 🎯 Catch to mark it as caught — it won't trigger notifications again. Click again to close.
History tab

Every notification that fires is logged here with callsign, timestamp and flight details. The last 100 entries are stored. Use the Clear button to wipe the history.

The 🎯 Caught aircraft dropdown shows all aircraft you have marked as caught. You can release individual aircraft or clear the entire list.

Settings
RadiusChoose 25, 50 or 100 km, or set a custom value with the slider
Hide ground trafficGlobal setting — suppresses both notifications and Live tab entries for aircraft on the ground
NotificationsToggle desktop notifications on or off. Polling continues either way.
SoundChoose an alert sound and volume. A preview plays when you select a sound.
ContentChoose which details appear in each notification: aircraft type, altitude, speed, route and/or direction
Startup tabChoose which tab opens when you launch the extension. Set it to a fixed tab or to Last used to always return to where you left off.
BackupExport all settings (alerts, location, radius, notification preferences) to a JSON file. Import restores everything at once. Old alerts-only backups are still supported.
FAQ
I'm not getting notificationsCheck that desktop notifications are enabled in Settings, and that Chrome notifications are allowed in your OS settings.
I get a 429 errorThe API rate limit was hit. The Live tab will show cached data. Wait a minute before refreshing.
Do alerts work when Chrome is closed?No — the background worker requires Chrome to be running, but no tab needs to be open.
Why only one notification per aircraft?Once a match is in range it stays tracked. When it leaves your radius it's forgotten, so you'll get a fresh alert if it returns.
I have multiple alerts and get duplicate notificationsThis is fixed — each aircraft triggers at most one notification per visit, regardless of how many alerts match it.
